OVERVIEW:

This project explores the relationship between the rise of Large Language Models and changes in Stack Overflow activity.
Using SQL to extract Stack Overflow data, I analysed trends in programming language related questions and visualized them
using Python and Mathplotlib.


The analysis revealed a noticeable decline in Stack Overflow question volume that coincides with the rapid adoption of AI coding assistants
and conversational LLMs such as ChatGPT.


FEATURES:

1.Data collection from Stack Overflow using SQL.

2.Analysis of question volumes across multiple programming languages as of May 2026.

3.Data visualisation using Matplotlib.

4.Trend comparison between Stack Overflow activity and the growth of LLM usage.

5.Analysis of matplotlib downloads over the last 10 years.


RESULTS:

The visualisations indicate a very strong negative correlation between the rise of LLM based coding assistants and Stack Overflow question volume.
As AI tools became more widely adopted, the number of programming-related questions on stack overflow dramatically decreased. The decrease is most noticeable
after Dec 2023, when ChatGPT was released.


Despite the decline in Stack Overflow activity, Python ecosystem metrics such as Matplotlib downloads show substantial growth during the same period. 
This suggests that reduced Stack Overflow activity does not imply reduced programming activity.
Matplotlib user activity results are consistent with the hypothesis that developers are relying more on alternative sources,
such as AI assistants, to solve coding related doubts.


LIMITATIONS:

It is important to note that correlation does not necessarily imply causation.

Other factors may have also contributed to the decline of Stack Overflow site traffic.

Package downloads are only a rough approximation for developer activity since automated systems may also generate downloads.
